# Стеганография CTF Challenge

Интерактивный веб-сайт в стиле хакинга с 5 заданиями по стеганографии.

## Описание

Сайт содержит 5 последовательных заданий по стеганографии. Каждое задание содержит ключ к следующему, а последнее задание содержит финальный флаг.

## Задания

1. **Задание 1**: Поиск букв по позициям в тексте
2. **Задание 2**: Шифр Цезаря (сдвиг на 3 позиции)
3. **Задание 3**: Расшифровка ASCII кодов
4. **Задание 4**: Комбинация техник (переворот + сдвиг)
5. **Задание 5**: Финальный флаг (комбинация всех техник)

## Использование

Просто откройте `index.html` в современном браузере. Все задания отображаются прямо на странице.

## Решения

### Задание 1
Найдите буквы в тексте по указанным позициям. Позиции считаются только букв и цифр (без пробелов и знаков препинания).

**Ответ:** `СКРЫТОЕ`

### Задание 2
Расшифруйте текст, закодированный шифром Цезаря со сдвигом на 3 позиции вперед. Для расшифровки сдвиньте каждый символ на 3 позиции назад.

**Ответ:** `KEY_STEG0_2`

### Задание 3
Преобразуйте ASCII коды в символы. Каждое число соответствует одному символу.

**Ответ:** `KEY_STEG0_3`

### Задание 4
Сообщение было закодировано в два этапа:
1. Строку перевернули (обратный порядок)
2. Каждый символ сдвинули на 1 позицию вперед

Для расшифровки выполните операции в обратном порядке:
1. Сдвиньте каждый символ на 1 позицию назад
2. Переверните строку

**Ответ:** `KEY_STEG0_4`

### Задание 5
Флаг был закодирован в три этапа:
1. Строку перевернули (обратный порядок)
2. Каждый символ сдвинули на 2 позиции вперед
3. Результат преобразовали в ASCII коды

Для расшифровки выполните операции в обратном порядке:
1. Преобразуйте ASCII коды в символы
2. Сдвиньте каждый символ на 2 позиции назад
3. Переверните строку

**Ответ:** `FLAG{ST3G4N0GR4PHY_MA5T3R_2024}`

## Ответы на задания

- **Задание 1:** `СКРЫТОЕ`
- **Задание 2:** `KEY_STEG0_2`
- **Задание 3:** `KEY_STEG0_3`
- **Задание 4:** `KEY_STEG0_4`
- **Задание 5 (Флаг):** `FLAG{ST3G4N0GR4PHY_MA5T3R_2024}`
